Language used: Java
to compile all three files, use : make 



to run the brute force approach, use:    java program1 -k knapsack01.txt

to run the refined DP approach, use:     java program2 -k knapsack01.txt

to run the backtracking approach, use:   java program3 -k knapsack01.txt


Note : please execute program 1 first as it will generate the input file knapsack01.txt and without that input file, program2 or program3 won't execute.
